# Agentic Policy Optimization via Instruction-Policy Co-Evolution

- PDF: https://arxiv.org/pdf/2512.01945
- Code: https://github.com/cambridgeltl/inspo

## Summary

The paper proposes a training framework in which both the model policy and the instruction (prompt) are optimized jointly using reinforcement learning. Instead of fixing a single instruction, the method maintains a set (or population) of candidate instructions. During training, an instruction is sampled, combined with an input (e.g., a question), and passed to the model to generate a trajectory (reasoning steps and final output). A reward is then computed based on the outcome, and this reward is used both to update the model policy and to evaluate the quality of the instruction.

Instructions are updated through a selection-and-refinement process. Instructions that consistently lead to higher rewards are retained, while weaker ones are removed. New instructions are generated by analyzing low-reward trajectories: the system uses an LLM to reflect on failures and propose improved instructions that better guide the model’s behavior. This creates an evolving pool of instructions that adapts over time based on performance.

In parallel, the model policy is trained with reinforcement learning to maximize expected reward under the current instruction distribution. The overall process forms a feedback loop: instructions shape the model’s behavior, the resulting rewards evaluate both the policy and the instructions, and these signals are used to iteratively update both components throughout training.
